# 🛍️ E-Commerce con IA Generativa

Este proyecto es una plataforma de e-commerce que facilita la creación de publicaciones de productos usando inteligencia artificial generativa. El usuario solo debe subir una imagen, y el sistema genera automáticamente:

- Un título atractivo (test)
- Una descripción persuasiva  
- Una categoría sugerida  
- Etiquetas (tags) útiles para SEO  
- (Opcional) Una sugerencia de precio  

### 👥 ¿A quién va dirigido?

- Personas sin experiencia vendiendo online  
- Vendedores por redes sociales (Instagram, WhatsApp, etc.)  
- Emprendedores y negocios pequeños con muchos productos por subir  

### 🧠 ¿Cómo funciona?

1. El usuario sube una imagen del producto  
2. Un modelo de visión por computador (como BLIP) genera una descripción base  
3. Un modelo de lenguaje (como GPT-4o) refina y completa la publicación  
4. El usuario puede editar o publicar directamente  

### 🔧 Tecnologías clave

- **BLIP** (image captioning – Hugging Face API gratuita)  
- **GPT-4o** (generación de texto con visión – OpenAI)  
- Alternativas: Vertex AI (PaLM), Claude, modelos open-source  

### 🎯 Objetivo

Reducir el tiempo y esfuerzo para publicar productos online y democratizar el acceso al comercio electrónico mediante IA.
