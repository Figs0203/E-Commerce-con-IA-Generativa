# 🛍️ E-Commerce con IA Generativa

Este proyecto es una plataforma de e-commerce que facilita la creación de publicaciones de productos usando inteligencia artificial generativa. El usuario solo debe subir una imagen, y el sistema genera automáticamente:

- Un título atractivo
- Una descripción persuasiva  
- Una categoría sugerida  
- Etiquetas (tags) útiles para SEO  
- (Opcional) Una sugerencia de precio  

### 👥 ¿A quién va dirigido?

- Personas sin experiencia vendiendo online  
- Vendedores por redes sociales (Instagram, WhatsApp, etc.)  
- Emprendedores y negocios pequeños con muchos productos por subir que desean automatizar este tedioso proceso
